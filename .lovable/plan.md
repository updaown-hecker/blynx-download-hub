
# Blynx Browser — Download & Marketing Page

## Overview
A polished, Apple-inspired marketing and download page for Blynx Browser — an Electron-based web browser. The page combines a dramatic dark hero section with clean light content sections, echoing the style of Apple product pages.

---

## Page Sections (top to bottom)

### 1. Navigation Bar
- Sticky top bar with the Blynx logo (user will upload), nav links (Features, Download, About), and a prominent "Download" CTA button
- Frosted glass/blur effect background (Apple-style)

### 2. Hero Section (Dark)
- Full-width dark gradient background with subtle glow effects
- Large bold headline: "Browse the web, reimagined."
- Subheading highlighting key value props (fast, modern, customizable)
- Platform download buttons (Windows, macOS, Linux) with OS icons
- A large browser screenshot/mockup as the centerpiece visual

### 3. Features Grid (Light)
- Clean white background section
- 2x3 grid of feature cards with icons highlighting:
  - Tab Management
  - Built-in Ad Blocker
  - Custom Protocols (blynx://)
  - Extension Support
  - Dark/Light Mode
  - Keyboard Shortcuts
- Each card has an icon, title, and short description
- Subtle hover animations

### 4. Screenshot Showcase
- Large, high-quality browser screenshot(s) with Apple-style rounded corners and shadow
- Can include multiple screenshots in a carousel or side-by-side layout

### 5. Download Section
- Clear call-to-action section with download buttons for each platform
- Shows current version number, file sizes
- User will provide real download URLs for Windows (.exe), macOS (.dmg), and Linux (.AppImage/.deb)

### 6. Footer
- Minimal footer with links to GitHub repo, license info, and support
- "Built with Electron" badge

---

## Design Style
- **Apple-inspired**: Large typography, generous whitespace, smooth scroll animations, frosted glass effects
- **Dark hero → light content**: Dramatic contrast between the hero and feature sections
- **Typography**: Clean sans-serif, large headings, readable body text
- **Animations**: Subtle fade-in on scroll effects for sections
- **Responsive**: Fully mobile-friendly layout

## User Action Needed
- Upload the Blynx Browser logo/icon
- Provide the actual download URLs for each platform (Windows, macOS, Linux)
- Optionally provide browser screenshots to showcase
